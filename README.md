# todo-list-js
A simple and interactive **To-Do List web application** built using **HTML, CSS, and JavaScript**. 
This project helps users add tasks, mark them as completed, delete tasks, and see a live task counter.

Features :
1) Add new tasks
2) Mark tasks as completed
3) Delete tasks
4) Live task counter
5) Clean and super interfaces
6) Fully dynamic using JavaScript DOM manipulation

Technologies Used

- **HTML** – Structure of the webpage  
- **CSS** – Styling and layout  
- **JavaScript** – Logic, DOM manipulation, and event handling
